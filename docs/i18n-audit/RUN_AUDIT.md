# Инструкция запуска аудита локализации

## Как запустить оркестратор

### Автоматический режим (рекомендуется)

Отправьте следующий промпт Claude:

```
Запусти оркестратор аудита локализации по плану docs/i18n-audit/ORCHESTRATOR.md

Порядок выполнения:
1. Запусти агента для проверки РУССКОГО языка по промпту docs/i18n-audit/prompts/PROMPT_RU.md
2. Дождись завершения и прочитай отчёт
3. Запусти агента для проверки УЗБЕКСКОГО языка по промпту docs/i18n-audit/prompts/PROMPT_UZ.md
4. Дождись завершения и прочитай отчёт
5. Запусти агента для проверки КИРГИЗСКОГО языка по промпту docs/i18n-audit/prompts/PROMPT_KY.md
6. Дождись завершения и прочитай отчёт
7. Запусти агента для проверки ТАДЖИКСКОГО языка по промпту docs/i18n-audit/prompts/PROMPT_TG.md
8. Дождись завершения и прочитай отчёт
5. Запусти агента для проверки АНГЛИЙСКОГО языка по промпту docs/i18n-audit/prompts/PROMPT_EN.md
9. Дождись завершения и прочитай отчёт
10. Сформируй финальный отчёт docs/i18n-audit/reports/FINAL_REPORT.md

После каждого агента обнови статус в ORCHESTRATOR.md
```

### Ручной режим (по одному языку)

Для проверки конкретного языка:

```
Выполни аудит локализации для [ЯЗЫКА] по промпту docs/i18n-audit/prompts/PROMPT_[CODE].md
```

Где `[CODE]` = RU, UZ, KY, TG, EN

---

## Ожидаемые результаты

После завершения в `docs/i18n-audit/reports/` появятся:

```
reports/
├── REPORT_RU.md      # Отчёт по русскому
├── REPORT_UZ.md      # Отчёт по узбекскому
├── REPORT_KY.md      # Отчёт по киргизскому
├── REPORT_TG.md      # Отчёт по таджикскому
├── REPORT_EN.md      # Отчёт по английскому
└── FINAL_REPORT.md   # Сводный отчёт
```

## Что делает каждый агент

1. **Читает** файл локализации (~2400 строк)
2. **Сравнивает** с эталоном (en.json)
3. **Проверяет** использование ключей в компонентах
4. **Анализирует** качество переводов:
   - Профессиональность терминологии
   - Грамматика
   - Стилистика
   - Контекстуальность
5. **Исправляет** проблемы прямо в файле
6. **Создаёт** детальный отчёт

---

## После завершения аудита

1. Просмотрите `FINAL_REPORT.md`
2. Проверьте изменения: `git diff apps/frontend/src/locales/`
3. При необходимости откатите: `git checkout apps/frontend/src/locales/[file].json`
4. Сделайте коммит: `git add . && git commit -m "fix(i18n): audit and fix localization"`
