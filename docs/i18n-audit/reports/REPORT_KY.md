## Особенности киргизского перевода

### Положительные стороны:
1. **Кириллица** - Используется официальный алфавит Кыргызстана
2. **Естественные формулировки** - Большинство переводов звучат естественно для носителя
3. **Корректные аффиксы** - Правильное использование киргизских падежных окончаний (-га, -ке, -дан и т.д.)
4. **Уместные заимствования** - Российские юридические термины (РВП, ВНЖ, ИИМ, ФССП) сохранены для понимания документов
5. **Консистентность терминов** - После исправлений все термины используются единообразно

### Рекомендации на будущее:
1. При добавлении новых ключей использовать РВП/ВНЖ вместо УЖР/ЖТК для консистентности
2. Избегать калькированных переводов типа "рас кылуу" - использовать "расмийлештирүү"
3. Сохранять российские аббревиатуры документов (ФССП, ИИМ, МММ) для понимания в контексте российской миграционной системы
4. Использовать глоссарий терминов при добавлении новых переводов
