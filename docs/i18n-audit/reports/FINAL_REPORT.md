## Ключевые улучшения по языкам

### Русский (ru) — Качество: ОТЛИЧНО
- Исправлены проблемы с множественным числом (`{days} дня` → `{days} дн.`)
- Удалены emoji из UI-текстов
- Русская локализация изначально была высокого качества

### Узбекский (uz) — Качество: ХОРОШО → ОТЛИЧНО
- **Терминология**: `migrantlar` → `muhojirlar` (официальный узбекский термин)
- **Аббревиатуры**: ЕАЭС → YOII, ФССП → FSSQ, ИНН → STIR
- Добавлены ключи QR-кода и dashboard.urgent
- Удалены emoji из 14 мест

### Киргизский (ky) — Качество: НЕПОЛНЫЙ → ХОРОШИЙ
- **Добавлено 53 ключа** — секция `checklist` была неполной
- Стандартизированы аббревиатуры (УЖР/ЖТК → РВП/ВНЖ)
- Исправлены калькированные переводы

### Таджикский (tg) — Качество: НЕПОЛНЫЙ → ХОРОШИЙ
- **Добавлен 51 ключ** — секция `checklist` была неполной
- Улучшена юридическая терминология
- Добавлены пояснения к русским аббревиатурам (СНИЛС, ИНН)
- Исправлена опечатка МФС → МФЦ

### Английский (en) — Качество: ХОРОШО → ОТЛИЧНО
- **28 терминологических исправлений**: русские аббревиатуры → английские термины
  - MVD → migration office/authorities
  - RVP → Temporary Residence Permit (TRP)
  - VNJ → Residence Permit
  - GIBDD → traffic police
- Применён консистентный Title Case для заголовков
- Улучшены формулировки для естественного английского

---

## Общие проблемы, найденные во всех локализациях

### 1. Структурные
- Секция `checklist` была неполной в ky и tg (~50 ключей)
- Секция `profile.qrCode` отсутствовала в uz, ky, tg

### 2. Терминологические
- Непоследовательное использование аббревиатур (ЕАЭС vs EAEU vs локальные)
- Русские аббревиатуры (МВД, ФССП, ИНН) не адаптированы для целевых языков

### 3. Стилистические
- Emoji в UI-текстах (удалены)
- Калькированные переводы с русского
