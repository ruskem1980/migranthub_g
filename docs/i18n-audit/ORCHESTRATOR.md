## Структура отчётов

Каждый агент создаёт отчёт в `docs/i18n-audit/reports/`:

```
docs/i18n-audit/
├── ORCHESTRATOR.md          # Этот файл
├── prompts/
│   ├── PROMPT_RU.md
│   ├── PROMPT_UZ.md
│   ├── PROMPT_KY.md
│   ├── PROMPT_TG.md
│   └── PROMPT_EN.md
└── reports/
    ├── REPORT_RU.md
    ├── REPORT_UZ.md
    ├── REPORT_KY.md
    ├── REPORT_TG.md
    ├── REPORT_EN.md
    └── FINAL_REPORT.md
```

## Команда запуска оркестратора

```
Запусти проверку локализации по плану docs/i18n-audit/ORCHESTRATOR.md
```
