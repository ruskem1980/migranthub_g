# Задача: Аудит локализации — Русский язык (ru)

## Контекст

Проект MigrantHub — приложение для мигрантов в России.
- Целевая аудитория: мигранты из СНГ
- Файл локализации: `apps/frontend/src/locales/ru.json` (~2409 строк)
- Эталон структуры: `apps/frontend/src/locales/en.json`
- Русский — основной язык приложения

## Цель

1. Проверить профессиональность и корректность переводов
2. Проверить применение всех ключей в компонентах
3. Исправить найденные проблемы
4. Создать отчёт об изменениях

## Критерии качества перевода

### Профессиональность
- Юридическая терминология должна быть точной (патент, РВП, ВНЖ, миграционный учёт)
- Нет машинного/калькированного перевода
- Естественные формулировки для носителя языка
- Консистентность терминологии во всём файле

### Стилистика
- Уважительное обращение к пользователю
- Понятный язык без канцеляризмов
- Краткость (UI-тексты должны быть лаконичными)
- Без опечаток и грамматических ошибок

### Контекстуальность
- Переводы соответствуют контексту использования (кнопки, заголовки, сообщения)
- Плейсхолдеры {{variable}} сохранены корректно
- Pluralization (множественные формы) корректны для русского

## Шаги

### 1. Анализ структуры
```bash
# Сравнить ключи ru.json и en.json
```
- Найти отсутствующие ключи
- Найти лишние ключи

### 2. Проверка качества перевода
Проверить по секциям:
- `common` — общие элементы UI
- `auth`, `login`, `recovery` — авторизация
- `documents` — документы
- `services` — сервисы
- `exam` — экзамен
- `sos` — экстренная помощь
- `wizard` — визард легализации
- И все остальные секции

### 3. Проверка использования в компонентах
Найти все вызовы `t('ключ')` и проверить:
- Ключ существует в ru.json
- Перевод уместен в контексте компонента

### 4. Исправления
Внести корректировки прямо в файл `ru.json`

### 5. Отчёт
Создать `docs/i18n-audit/reports/REPORT_RU.md`

## Файлы для работы

- `apps/frontend/src/locales/ru.json` — проверить и исправить
- `apps/frontend/src/locales/en.json` — эталон структуры
- `apps/frontend/src/**/*.tsx` — проверить использование
